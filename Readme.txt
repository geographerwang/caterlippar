================“平差助手V1.5”使用说明===================

【毕业论文基于本版本】


V1.5版本更新说明:

    ◆软件的运行比V1.0版本更加稳定。

    ◆对整个软件进行了重构，降低各模块之间的耦合性，方便以后升级扩展。

    ◆更新了测试用例，之前由于手工计算错误，导致误差过大的近似平差算例得到更新；严密平差和高斯反算的XY坐标颠倒的情况被纠正。

    ◆输出为dxf文件时，将各点点名添加进dxf文件。

    ◆将展点图和导线图的图形比例尺屏幕显示参数设置为72PPI。


※1. 软件安装
    直接点击setup.exe安装即可，然后用注册机激活软件。
（注意：软件基于.Net Framework 4.0 环境，如果没有安装.Net Framework 4.0，软件会自行下载安装。光盘“设计”目录的3rdPart文件夹有.Net Framework 4的安装包。用户也可到微软官网下载，下载地址：http://www.microsoft.com/zh-cn/download/details.aspx?id=24872）


※2. 软件卸载
    点击“开始”――>找到“平差助手”文件夹――>点击“卸载平差助手”即可。也可以在“控制面板”中卸载。


※3. 测试用例
    为了方便用户熟悉软件，作者添加了测试用例在“测试用例”文件夹中。其中：
高斯坐标的坐标带号为了与MapGIS等商业软件保持一致，被去除掉。测试数据的中央经线均为111，分带为3度带。高斯反算数据同样适用于坐标换带，参数同前面一样。
具体操作请参见软件的“帮助”。

※4. 文件输出
    Word和Excel的文件输出部分配合“MS Office 2010”进行操作。否则输出失败。



    如果软件异常退出，请查看程序安装目录的“Error.log”文件的日志记录，进行调试或者将错误和测试数据发送至作者邮箱。
